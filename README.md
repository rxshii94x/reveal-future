# ⚜️ REVEAL:FUTURE

### The Record Endures. Every Fragment Still Remembers.

REVEAL:FUTURE is a cinematic scroll-driven digital archive where users explore preserved memories through an immersive futuristic museum experience guarded by the Chrome Guardian.

Designed as an interactive digital installation rather than a traditional website, the experience blends narrative storytelling, luxury motion design, cinematic transitions, and scroll-driven exploration to create a journey through memory, preservation, time, and human legacy.

## 🏛️ Experience Overview

Visitors enter a protected archive and progress through a series of evolving chapters where preserved fragments of memory gradually reveal themselves.

The experience is designed to feel closer to a digital museum, cinematic installation, and interactive archive than a conventional website.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/rxshii94x/reveal-future.git
```

### Navigate To Project

```bash
cd reveal-future
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

## 🎨 Design Philosophy

REVEAL:FUTURE explores a simple question:

**"What happens to memory when time itself becomes an archive?"**

The project was created as an experiment in immersive web storytelling where motion, atmosphere, interface design, and narrative become a single experience.
